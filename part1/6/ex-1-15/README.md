# Exercise 15, part 1, Docker UOH

Just a dummy react app that you can run as a Docker container. 

## Installation

**Requirements:** `docker` and `docker-cli`

```sh
docker run -p 5000:5000 superjordi/ex-1-15
```


## Usage

**web url:** 
`http://localhost:5000/`
